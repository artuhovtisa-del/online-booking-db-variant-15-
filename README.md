## 📋 Описание предметной области

Система онлайн-записи в медицинский центр на вакцинацию. Поддерживает:
- Выбор вакцины с проверкой наличия на складе и срока годности
- Учёт возрастных ограничений (18+) и согласия родителей для несовершеннолетних
- Регистрацию побочных реакций после вакцинации
- Управление расписанием специалистов и статусами записей
